
Q1: Why might be consider using git?

Answer: We would consider using git because it helps handle source control between multiple people. If there is a team working on a project, multiple people can contribute to the project without interfering with each other's changes. Also git is useful to keep track of what changes have been made to a project.

Answer: Data science could be applied to the financial sector. There is more data today about financial markets than there has ever been before. Data science can help address this large collection of data and make into novel predictions about the future of certain companies.
